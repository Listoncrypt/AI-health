# HealthPulse AI

HealthPulse AI is a personalized health & fitness coach designed to help users track workouts, manage nutrition, scan for early health risks, and receive personalized coaching.

## Project Structure
- `backend/`: Node.js Express REST API with local SQLite database and mock AI/ML recommendation systems.
- `frontend/`: Vite + React single-page application with modern, responsive Vanilla CSS design.
